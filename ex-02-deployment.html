In this guided exercise, you will learn how to create a simple microservice app using a NodeJS and a MySQL containers. #Outcomes You should be able to create a basic Microservice Architecture. #Steps 1. Your instructor will provide you all the manifest files needed to build all microservices. 2. NodeJS Microservice - Replace the replicaSet by the Deployment controller - Create a new manifest file named: `nodejs-deployment.yml` - Deployment Name: `nodejs-deployment` - Number of replicas = `1` - Labels/Selectors = `nodejs-instance` - Apply the manifest file and check resources creation 3. Add more replicas to your nodeJS microservice - Number of replicas = `3` 4. Test your application
