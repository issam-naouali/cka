In this guided exercise, you will learn how to create a simple microservice app using a NodeJS and a MySQL containers. #Outcomes You should be able to create a basic Microservice Architecture. #Steps 1. Your instructor will provide you all the manifest files needed to build all microservices. 2. Mysql Database - Configure a Cluster IP Service for Mysql Instance 3. NodeJS Microservice - Configure a node Port Service for nodeJS microservice - Use Mysql Service in MYSQL_HOST instead of using Pod IP 3. Add more replicas to your nodeJS microservice - Number of replicas = `5` 4. Test your application from outside the cluster
